This folder contains the source code for the bootloader provided with Darron Baida's Xiphos Board.
It is a modified copy of the "AVRPROG compatible bootloader for ATMEL ATmega Controllers" 
written by Martin Thomas, version 0.85 (Dec 3, 2008). The original source code was downloaded from:
http://www.siwawi.arubi.uni-kl.de/avr_projects/#avrprog_boot
For reference, a copy of the original source code is retained in the adjacent "Original Sources" folder.
Bootloader source code modified to support Xiphos Board by Patrick McCarty in 2009.

You are free to modify this bootloader code however you see fit and redistribute it, provided you 
include this README.txt file in the distribution as attribution (to which you may add your own 
attribution for your modifications).
